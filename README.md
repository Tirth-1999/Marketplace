# Marketplace

Simple local marketplace: product photos, prices, filters, and WhatsApp contact.

## Run locally

```bash
npm install
npm run dev -- --hostname 127.0.0.1
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000).
